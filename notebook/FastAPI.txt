带有 @ 符号的语法叫做 装饰器（Decorator）
不改动原代码的前提下，给函数动态增加新功能
例：@app.get("/")

.get()  HTTP GET 请求（通常用于获取数据，比如你在浏览器输入网址敲回车）
("/") 代表根目录
("/api/upload") 代表对应目录

.post() 用于向后端送数据
1. 接收纯文本/JSON 数据（最常用）

def chat_with_ai(data: ChatQuery): 
    # FastAPI 会自动把前端发的 JSON 转化为 data 对象
    return {"message": f"你问了：{data.query}，限制字数：{data.max_tokens}"}

2. 接收文件

@app.post("/api/upload")
def upload_pdf(file: UploadFile = File(...)):
    # file.filename 能拿到文件名
    # file.read() 能读到文件二进制内容
    return {"filename": file.filename}

3. 接收 URL 路径参数或查询参数

@app.post("/api/user/{user_id}")
def update_user_status(user_id: int, status: str):
    return {"user_id": user_id, "status": status}
